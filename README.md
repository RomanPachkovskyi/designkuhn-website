# DesignKuhn

Portfolio website for a custom clothing and textile design studio.

## Quick Start

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Tech Stack

- **Backend:** Express.js + EJS
- **Styling:** Bootstrap 5.3.3
- **i18n:** German (de), English (en)

## Documentation

- [PROJECT.md](docs/PROJECT.md) — Project documentation and changelog
- [MIGRATION_NEXTJS.md](docs/MIGRATION_NEXTJS.md) — Next.js migration plan

## Deployment

- **Production:** https://designkuhn-website.vercel.app
- **Platform:** Vercel (serverless)

## Environment Variables

Create `.env` file:

```
SMTP_USER=your_mailtrap_username
SMTP_PASS=your_mailtrap_password
PORT=3000
```
